
# Основная идея
Основываясь на геометрических формулах, вычесляет значение площади и периметра треугольника, круга, квадрата и прямоугольника.

# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle: S = ah/2

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
- Triangle: P = a + b + c


# Файлы
## circle.py
### area(r)
Принимает радиус круга r и возвращает его площадь
Пример:area(5) = 78.53981633974483
### perimeter(r)
Принимает радиус круга r и возвращает его периметр
Пример: perimeter(5) = 31.41592653589793
## rectangle.py
### area(a,b)
Принимает длинны 2 сторон треугольника и возвращает его площадь
Пример:area(3,5) = 15
### perimeter(a,b)
Принимает длинны 2 сторон прямоугольника и возвращает его периметр
Пример: perimeter(3,5) = 16
## square.py
### area(a)
Принимает длинну стороны квадрата а и возвращает площадь квадрата
Пример: area(5) = 25
### perimeter(a)
Приниммет длинну стороны a и возвращает периметр квадрата
Пример: perimeter(5) = 20
## triangle.py
### area(a,h)
Принимает длинну стороны и длинну высоты треугольника(a,h) и возвращает площадь
Пример: area(4,5) = 10
### perimeter(a,b,c)
Принимает длинны 3 сторон треугольника(a,b,c) и возвращает его периметр'
Пример: perimeter(3,4,5) = 12
